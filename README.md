# Сложно сосредоточиться

Адаптивный лендинг о проблемах концентрации внимания и способах их решения.

## О проекте

Проект выполнен в рамках учебной задачи. Основная цель — продемонстрировать навыки адаптивной вёрстки, работы с CSS-переменными и поддержки светлой/тёмной темы.

## Функциональность

- Полностью адаптивная вёрстка (mobile-first подход)
- Светлая и тёмная темы оформления
- Автоматическое переключение темы в зависимости от системных настроек
- Ручное переключение темы через кнопки
- Декоративные элементы (уголки, плашка REC)
- Галерея изображений со сложной грид-сеткой

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, Clamp, медиазапросы)
- CSS-переменные для тем
- JavaScript (переключение тем)
